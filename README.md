1. Install nvm https://github.com/nvm-sh/nvm#installation-and-update
2. nvm install v12.14.0
3. git clone https://github.com/vladislavmamchich/insta_photo.front.git
4. cd insta_photo.front
5. npm install
6. cp .env.example .env
7. npm start
